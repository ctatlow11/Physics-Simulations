# Physics Simulation Projects

## Overview
This project contains several physics-based simulations written in Python, including motion, acceleration, voltage models and torque models.

## Projects Included
- Bouncing Ball Simulation
- Car Acceleration Model
- Torque Simulation
- Simulations: Voltage modeling simulation

## How to Run

1. Make sure Python is installed (Python 3 recommended)

2. Clone the repository:
git clone https://github.com/ctatlow11/Physics-Simulations.git

3. Navigate into the project folder:
cd Physics-Simulations

4. Run a simulation:
python "Bouncing Ball Project - 1.py", etc.

## Concepts Demonstrated
- Kinematics
- Numerical simulation
- Physics modeling
